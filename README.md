# Responsive Bakery Website
A modern, responsive, multi-page bakery website built using HTML, CSS,Bootstrap and JavaScript, designed to showcase real-world UI design, layout structuring, and frontend development skills.
This project demonstrates my ability to convert design ideas into a complete, user-friendly web experience suitable for small businesses.

**Project Objective
The main objective of this project is to:**
Design a professional business website for a bakery
Practice real-world frontend development
Implement responsive layouts for mobile, tablet, and desktop
Add interactive UI elements using JavaScript
Follow proper project structure and clean code practices

**Technologies Used**
**HTML5**
Semantic tags for accessibility and SEO
Structured multi-page layout
**CSS3**
Flexbox & Grid for layout design
Custom color variables for consistent theme
Animations and hover effects
Media queries for responsiveness
**JavaScript** 
DOM manipulation
Event handling
Dynamic UI updates (tabs, navigation, scroll effects)

**Libraries & Tools**
Font Awesome – Icons
Google Fonts – Typography
VS Code – Development
Git & GitHub – Version control

**What I Learned..?**
Building real-world multi-page layouts
Responsive UI using Flexbox & Grid
Reusable CSS components
JavaScript DOM manipulation
Improving UI/UX for business websites
